# Requirements

- "Big Blue nerd plus" Nerd font.
- `gpsd`

# Custom plugins

### `custom/docker-status`

Shows the status of the docker daemon. You can hover it to show which containers are running.

### `custom/pacman-status`

Shows the status of pacman registry.
