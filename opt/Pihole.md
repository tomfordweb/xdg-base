# Installation

On a fresh and up to date ubuntu install.

1. [Expose ports](https://docs.pi-hole.net/main/prerequisites/)
2. [Install as root](https://docs.pi-hole.net/main/basic-install/)
3. Setup DNS
4. `sudo usermod -aG pihole $USER`
5. [Setup WireGuard](https://docs.pi-hole.net/guides/vpn/wireguard/)
